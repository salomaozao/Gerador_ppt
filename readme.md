Projeto para geração de relatórios PPTX com base em dados de pesquisas.

## Funcionalidades
- Upload de arquivos CSV ou Excel
- Personalização de cores
- Configuração de visualizações
- Exportação de relatórios PPTX

## Requisitos
- R >= 4.4
- shiny
- bslib
- shinyjs
- waiter
- shinyalert
- dplyr
- tidyr
- purrr
- colourpicker
- reactable
- ggplot2
- officer
- rvg
- mschart
- flextable
